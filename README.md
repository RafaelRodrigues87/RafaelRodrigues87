## 🚀 Projetos em Destaque

### 💰 GranaControl
> Sistema completo de controle financeiro pessoal

**Stack:** Java 21 · Spring Boot 3 · Spring Security · JWT · MySQL · React · Docker

- 🔐 Autenticação JWT com recuperação de senha via e-mail (Gmail SMTP)
- 🏦 Gerenciamento de contas bancárias com controle de saldo automático
- 📊 Dashboard com gráfico de receitas vs despesas (últimos 30 dias)
- 🎯 Metas de saldo personalizáveis com barra de progresso
- 🐳 Backend e banco containerizados com Docker Compose

[![Repo](https://img.shields.io/badge/Ver_Repositório-1e293b?style=for-the-badge&logo=github&logoColor=white)](https://github.com/RafaelRodrigues87/granacontrol)
